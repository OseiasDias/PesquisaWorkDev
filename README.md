
# Quadro de Empregos – Frontend

Este projeto corresponde à **versão 1** de um **Quadro de Empregos**, desenvolvido com foco em aprendizagem e prática de **JavaScript moderno** e **ReactJS**.

A aplicação consome dados diretamente de um **arquivo JSON local**, simulando uma fonte de dados real, permitindo listar e visualizar oportunidades de emprego de forma simples e organizada.



## Objetivo do Projeto

* Praticar **JavaScript** e **ReactJS**
* Trabalhar com **componentização**
* Consumir dados a partir de um **JSON local**
* Estruturar um projeto frontend moderno
* Preparar a base para futuras integrações com APIs reais


## Tecnologias Utilizadas

* **ReactJS**
* **JavaScript (ES6+)**
* **Vite**
* **HTML5**
* **CSS3**
* **JSON** (fonte de dados local)

##  Como Executar o Projeto Localmente

1. Clone o repositório:

   
   git clone https://github.com/seu-usuario/quadro-de-empregos.git
   

2. Acesse a pasta do projeto:


   cd quadro-de-empregos
 

3. Instale as dependências:

   
   npm install


4. Execute o projeto:

   npm run dev


5. Acesse no navegador:

 
   http://localhost:5173
